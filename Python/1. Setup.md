# Python Environment Setup Guide
1. Install anaconda
2. Create a folder having all library files
3. Create a new conda environment inside the folder using cmd

   conda create -p venv python==3.12
4. Activate the environment 

   conda activate venv/
5. Create a requirements.txt file inside the folder and add all the required libraries
6. Install all the libraries using the command

   pip install -r requirements.txt
7. To deactivate the environment use

   conda deactivate
8. To remove the environment use

   conda remove --prefix venv/ --all
9. To export the libraries installed in the environment to a requirements.txt file use
   
    pip freeze > requirements.txt
10. To install a specific library use

    pip install library_name==version
11. To uninstall a library use

    pip uninstall library_name
